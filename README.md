# Meu Projeto do TCC

## Fluxo de Navegação e UX
Este projeto segue o padrão MVC para garantir que a interface (View) seja
independente da lógica (Controller).

### Menu Principal
1. **Início:** Visão geral do sistema.
2. **Dashboard:** Painel principal com estatísticas.
3. **Configurações:** Edição de perfil e preferências.
4. **Login:** Tela de autenticação onde o usuário insere suas credenciais. 


### Fluxo de Ação Principal
Para realizar a função [NOME DA FUNÇÃO], o usuário percorre:
1. **Tela de Login:** Autenticação segura.
2. **Página Inicial:** Clique no botão de ação rápida.
3. **Formulário de Entrada:** Preenchimento de dados validados.
4. **Feedback Visual:** Mensagem de sucesso ao finalizar.

### Acessibilidade e Otimização
* **Contraste:** Utilizamos uma paleta de cores com alto contraste para facilitar
a leitura.
* **Navegação:** Implementamos nomes intuitivos nos botões (ex: em vez de
"OK", usamos "Salvar Alterações").
